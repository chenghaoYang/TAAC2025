02 — DATASET PIPELINE (dataset.py)
MyDataset — Training
1. Load seq → full_events[] + user_indices{}
2. For uid: reconstruct user sequence
3. Interleave item + user feature rows
4. Left-pad to maxlen=101
5. Generate pos (next real item)
6. Neg sample: random ≠ history items
7. fill_missing_feat() for all fields

Output per sample:
  seq[101], pos[101], neg[101]
  token_type[101] (1=item, 2=user)
  next_token_type[101]
  seq_feat[101], pos_feat[101], neg_feat[101]

1. Same loading logic as training
2. Cold-start handling: str feats → 0
3. Items > itemnum → masked to 0
4. Left-pad to maxlen=101
5. Keep raw user_id for output mapping

03 — MODEL ARCHITECTURE (model.py — BaselineModel)
Feature Fusion (feat2emb)
Item Feature Pipeline
item_emb(ID) → 32d
+
13 × sparse_emb → 32d each
+
mm_emb × Linear → 32d each
→ concat → Linear(32+13×32+6×32, 32) + ReLU
→ 32d item representation
User Feature Pipeline
user_emb(ID) → 32d
+
4 × sparse_emb → 32d each
+
4 × array_emb.sum → 32d each
→ concat → Linear(32+4×32+4×32, 32) + ReLU
→ 32d user representation
seqs_emb = item_repr + user_repr  (element-wise sum)   × √d
Sequence Encoder (log2feats) — SASRec Transformer
┌── repeat × num_blocks (default: 1) ──┐
LayerNorm(hidden) Pre-Norm
FlashMultiHeadAttention heads=1, dim=32, causal mask
+ Residual seqs = seqs + attn_out
LayerNorm(hidden) Pre-Norm
PointWiseFeedForward Conv1D → ReLU → Dropout → Conv1D
+ Residual seqs = seqs + ffn_out
└──────────────────────────────────────┘
LayerNorm → log_feats (B, maxlen, 32)
Attention Mask
Causal: lower-triangular (auto-regressive)
Padding: mask != 0 (valid positions)
Combined: causal & padding

mask values: 1 = item, 2 = user
Only positions where next == item contribute to loss
Position Encoding
Learned: Embedding(2 × maxlen + 1, 32)
pos = arange(1, maxlen+1) × (seq != 0)
seqs += pos_emb(pos)
Output Heads
Training: forward()
pos_logits = (log_feats · pos_emb).sum(-1)
neg_logits = (log_feats · neg_emb).sum(-1)
loss_mask = (next_token_type == 1)

loss = BCE(pos_logits[mask], 1)
     + BCE(neg_logits[mask], 0)
     + l2_emb × ‖item_emb‖
Inference: predict()
log_feats = log2feats(seq, mask, feat)
final_feat = log_feats[:, -1, :]

→ Take last position as user representation
→ Shape: (B, 32)
→ Used as query for ANN search
